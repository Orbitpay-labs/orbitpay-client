# Contributing To OrbitPay Client

Thanks for helping build OrbitPay Kit.

## Local Work

Open `index.html` in a browser and edit:

- `styles.css` for UI,
- `app.js` for interactions,
- `index.html` for structure.

## Contribution Ideas

- Freighter wallet adapter.
- Checkout embed component.
- Explorer links.
- Better mobile states.
- Testnet API status.

Keep PRs small and include screenshots for UI changes.
